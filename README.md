# calid-map

## Getting Started

```bash
pnpm install
pnpm dev
```

## Scripts

```bash
pnpm dev              # Development server
pnpm build            # Production build
pnpm check            # All checks (Oxlint + Oxfmt + Knip)
pnpm fix              # Auto-fix lint + format
pnpm test             # Run tests
pnpm storybook        # Storybook dev server
```
